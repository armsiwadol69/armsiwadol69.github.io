Class and ID Selector

การใช้ Class Selector --> การใช้ Selector แบบ Class การประกาศเริ่มต้นด้วย ( . ) ตามด้วยชื่อ Class โดยชื่อ Class สามาเป็นตัวอักษร ตัวเลข เครื่องหมาย - , _ เท่านั้น ห้ามมีช่องว่าง วิธีการใช้งาน - Class Selector เหมาะกับการใช้เมื่อต้องการแสดงผลรูปแบบนั้นซ้ำหลายๆ ครั้งกับ element หลายๆ อัน
การใช้ ID Selector -->การใช้ Selector แบบ ID การประกาศเริ่มต้นด้วย ( # ) ตามด้วยชื่อ ID โดยชื่อ ID สามาเป็นตัวอักษร ตัวเลข เครื่องหมาย - , _ เท่านั้น ห้ามมีช่องว่าง วิธีการใช้งาน - ID Selector เหมาะกับการใช้เมื่อต้องการแสดงผลกับ element เพียงตัวเดียว เนื่องจาก ID เป็น unique สามารถมีได้ 1 จุดต่อ document ถ้ามีการประกาศซ้ำจะแสดงผลเพียงครั้งเดียว
